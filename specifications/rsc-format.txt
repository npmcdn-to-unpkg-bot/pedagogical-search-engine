
Format specifications of a pedagogical [ReSourCe] (.rsc)
================================
The following data is handled
	(d1) metadata. e.g. Title, Publisher, Date, version, Authors, ..
	(d2) tocs
	(d3) descriptions

Format type: JSON
Structure
{
	() [optional] (metadata) object (d1)
	() [optional] (tocs) list (d2)
		() [optional] (label)
		() [optional] (children)
	() [optional] (descriptions) list (d3)
}

Notes about (tocs)
	() (label)s may or may not contain an indication about the Structure.
		e.g. "Chap 3: Advanced concept", or simply "Advanced concept"

	() (label)s structure indications have no constraints
		e.g. "Chap 3: ..", "Chapter 3: ..", "3) ..", "[Three] ..", ..

	() root (children) entry is not necessarily the resource title
		e.g. book or course

Common elements of (metadata)
	(source) Identify where does the resource come from
	(level) Level of expertise of the resource
		(v:fundamental) Negligeable prior knowledge
		(v:high-school) Fundamental prior knowledge
		(v:university) High-school prior knowledge
		(v:expert) University prior knowledge
	(title) Title of the resource
	(authors) Authors of the resource
	(partners) Partners of the resource
	(created) Creation date
		format: yyyy-mm-dd
	(update) Last update date
		format: as in (created)
	(keywords) Keywords associated with the resource
	(categories) Category labels associated with the resource
	(href) relative url to the resource
		note: Then the source must be well known!
	(miniature) path to a miniature img
	(screenshot) path to a screenshot of the course
	(domain) label stating the domain
	(subdomain) label describing a subdomain

Sample (.rsc) file
================================
{
	"metadata": {
		"Title": "..",
		"authors": [],
		..
	},
	"tocs": [
		{
			"children": [{
					"label": "Chapter 1: Introduction",
					"children": [..]
				}, {
					"label": "Basic concepts",
					"children": [..]
				}, {
					"children": [..]
				}, {
					"label": "A note about ..",
					"children": [{
							"label": ".."
						},{
							"label": ".."
						},{
							"label": ".."
						},
					]
				}
			]
		}
	]
}
